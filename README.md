# ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/dikshitsinhaofficial/portfolio.git
```

Navigate to project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
